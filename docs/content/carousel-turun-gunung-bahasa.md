### Halaman 3-4: 10 Langkah

1. Identifikasi kebutuhan AI Anda
   → Customer service? Inventori? Dokumen? Pilih 1.

2. Pilih model yang cocok
   → Tabel: Llama 3.2 (general), Phi-3 (ringan), Gemma (multilingual)

3. Download dan install Ollama
   → Gratis, satu command: curl -fsSL https://ollama.ai/install.sh | sh

4. Jalankan model pertama
   → ollama run llama3.2

5. Test dengan pertanyaan bisnis Anda
   → Contoh prompt untuk customer service, inventori, dll

6. Hubungkan ke WhatsApp (opsional)
   → Tools: n8n, Flowise, atau custom

7. Fine-tune untuk bahasa Anda (opsional)
   → Dataset kecil: 500-1000 contoh

8. Deploy di laptop/server kantor
   → Spesifikasi minimum: 8GB RAM, 10GB disk

9. Monitor dan evaluasi
   → Checklist: akurasi, kecepatan, biaya

10. Scale bertahap
    → Dari 1 use case ke banyak, dari 1 laptop ke server
